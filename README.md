# M6 Farmakoloji Sınav Analizi

AÜTF Dönem 3 Modül 6 farmakoloji çıkmış soru analizi.

🔗 **[Uygulamayı Aç](https://ylmzeminm-jpg.github.io/farmacikmis/)**

## Özellikler

- 149 ilaç, H1–H5 tüm sınav verileri
- Her ilacın TUS'ta nasıl sorulduğu (gerçek soru metinleri)
- Soru tipi analizi: Tuzak / Seçici / Antidot / FK / Mekanizma / Yan Etki
- Gruplara göre görünüm (Antiepileptik, Antiparkinson, vb.)
- Arama, filtreleme, sıralama
- Yıldızlama ve versiyon geçmişi
- Koyu tema, mobil uyumlu
